## 🎯 **QILISH KERAK BO'LGAN QADAMLAR:**

### 1️⃣ **GitHub Repository yarating**
```bash
git init
git add .
git commit -m "Smart Calculator Mobile App"
git branch -M main
git remote add origin https://github.com/USERNAME/smart-calculator.git
git push -u origin main
```

### 2️⃣ **GitHub Actions avtomatik build**
- Kodni GitHub ga yuklang
- **Actions** bo'limiga boring
- **Android APK** va **iOS** avtomatik build bo'ladi
- **Releases** bo'limida APK faylini yuklab oling
